# Resum del Projecte: Shôko Cinema

**Nom:** Shôko Cinema  
**Curs:** 2DAW 2025-2026  
**Integrants:** Amin  
**Objectiu:** Plataforma de venda d'entrades concurrent en temps real per a esdeveniments d'alta demanda.  

**Arquitectura:**
- **Frontend:** Nuxt 3 + Pinia (Composition API) per a una UI reactiva i Shôko-aesthetic. Sincronització via Socket.IO Client.
- **Backend:** Laravel (API REST) per a persistència i validació de negoci.
- **Real-Time:** Node.js + Socket.IO per a la gestió de concurrència i bloqueig de seients.
